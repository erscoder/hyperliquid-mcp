# Usage Examples

Once configured, just ask Claude in natural language:

## Market Data (no wallet needed)

```
What's the current BTC price on Hyperliquid?
```
```
Show me the ETH orderbook top 5 levels
```
```
What are the last 24 hours of BTC candles (1h)?
```
```
What's the max leverage for SOL on Hyperliquid?
```
```
Which assets have the highest funding rates right now?
```

## Account (requires HL_WALLET_ADDRESS)

```
What are my open positions?
```
```
What's my total PnL across all positions?
```
```
Show me my open orders
```
```
What were my last 20 trades?
```

## Trading (requires HL_PRIVATE_KEY)

```
Buy 0.1 BTC at market price
```
```
Place a limit order to sell 1 ETH at $3500
```
```
Cancel all my open orders on SOL
```
```
Close my entire BTC position
```
```
Set my ETH leverage to 10x cross margin
```

## Combined analysis

```
Check my portfolio: show positions, PnL, and current funding rates for my coins
```
```
I have a long BTC position — what's the current funding rate and should I be worried?
```
